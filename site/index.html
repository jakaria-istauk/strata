Everything you need

A full admin client, not a query box

Data browsing, row editing, schema operations, raw SQL, and live server metrics — in one fast, themeable interface.

Data Browser

Sortable, searchable, paginated grids with foreign-key links, per-table column show/hide, and full-table CSV export.

Row CRUD

Slide-in drawer to view, edit, and insert rows. Per-field NULL toggles, server-side md5/sha hashing, transactional bulk delete.

Schema ops

Create and drop databases, build tables with a column builder, alter columns. Every destructive action is type-to-confirm.

SQL editor

Run arbitrary SQL with ⌘/Ctrl+Enter. Result grid vs affected-rows, query timing, and a localStorage query history.

Live dashboard

Server metrics — connections, uptime, questions, slow queries, DB and table counts and size — plus a query-type breakdown.

Stateless & private

Credentials and all app state live in your browser's localStorage. The PHP backend is stateless and holds no config or secrets.

Quick start

Drop the prebuilt bundle into any PHP host — XAMPP, WAMP, Herd, Valet — or run PHP standalone. No build step. Pick your stack →

  1. Download & unzipGrab strata.zip from the latest release and unzip it.
  2. Put it where PHP runsA web root (XAMPP/WAMP) or a parked dir (Herd/Valet) — or just serve it yourself.
  3. Open & connectVisit the URL, add a connection profile, start browsing.
# 1 — unzip into the Apache web root
unzip strata.zip -d htdocs/strata
#   C:\xampp\htdocs (Win) · XAMPP/htdocs (Mac)

# 2 — start Apache in the XAMPP Control Panel
#     (no php -S needed)

# 3 — open it
→ http://localhost/strata/

Apache already runs PHP — skip php -S. MySQL is the one bundled with XAMPP (host 127.0.0.1, user root).

# 1 — unzip into the WAMP web root
unzip strata.zip -d C:\wamp64\www\strata

# 2 — start WAMP, wait for the tray icon to go green

# 3 — open it
→ http://localhost/strata/

WAMP's Apache serves the PHP for you. Connect to WAMP's MySQL/MariaDB on 127.0.0.1:3306.

# 1 — unzip into a parked / linked directory
unzip strata.zip -d strata
cd strata

# 2a — Herd: drop the folder in your parked path, or
# 2b — Valet: link it
valet link strata

# 3 — open the auto domain
→ http://strata.test/

Herd & Valet auto-serve the folder over PHP — no server command. Use a non-HTTPS http:// link unless you ran valet secure.

# 1 — unzip the bundle
unzip strata.zip -d strata
cd strata

# 2 — serve it (PHP 8.x + pdo_mysql, nothing else)
php -S 127.0.0.1:8000

# 3 — open it
→ http://127.0.0.1:8000/

Zero install beyond PHP itself — point Strata at any reachable MySQL/MariaDB.
